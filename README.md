# EmguCVExample

This project demonstrate the YOLO object detection algorithm and show images on WPF UI with EmguCV.

*Code example include:*
* WPF UI data binding
  * image
* WPF data grid usage
  * show a list of object data
* WebCam
  * Capture image
  * Record video
  * Modify camera parameter
* EmguCV 
  * Read, show images
  * YOLO
  * FindChessboardCorners

## Including Package
* EmguCV 4.4.0
* Accord 3.8
